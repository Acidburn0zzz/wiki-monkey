Wiki Monkey is a MediaWiki-compatible bot and editor assistant that can be used
directly within wiki pages in the browser. It is available for Mozilla Firefox
(through [Scriptish](https://github.com/scriptish/scriptish) or
[Greasemonkey](https://github.com/greasemonkey/greasemonkey)) and Google Chrome
(through [Tampermonkey](http://code.google.com/p/tampermonkey/)) and can be
easily extended through the creation of plugins, exploiting its high-level API.

started.
